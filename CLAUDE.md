# pure-vue-stack-presentation

A static, CDN-only Reveal.js deck about our move to a pure Vue 3 stack
(no Nuxt): UI-framework decision, component classification, where logic
lives (composables / stores / utils), and Labs + design tokens — all in an
agent-driven development flow. English slides; Farsi narration in `STORY.md`.

No build step. Never add package.json, bundlers, or node_modules.

## Project skills

- `.claude/skills/deck-authoring/SKILL.md` — read BEFORE adding or editing
  slides, deck styles, Mermaid diagrams, or `STORY.md` narration. Covers the
  CDN-only rules, slide text budgets, the Mermaid-in-Reveal lazy-render fix,
  the STORY.md format, and the verification checklist.
